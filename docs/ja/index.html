lhasa

FOSS（ISC）の LHA / LZH / PMarc デコーダ、単一の自己完結型静的バイナリで配布。

lhasa は内部的に fragglet/lhasa を vendoring——ISC ライセンスのライブラリ + CLI、作者 Simon Howard。 デコーダ自体はメンテせず、ゼロ依存の静的バイナリにして配布するのが このリポジトリの仕事です。

macOS / Linux / Windows インストール 4 つの主要コマンドを見る

~200 KB 静的バイナリ · すべての Linux / macOS / Windows で libc / zlib / libiconv 不要。
