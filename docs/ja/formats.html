lhasa が読めるもの — 3 つのファミリー、10 のフォーマット。

.lzh — LHA / LZH / LHarc (1988-)

LHA（旧名 LHarc）は 1988 年に Haruyasu Yoshizaki が作った。 使うのは LZARI / LZHUF。 圧縮方式はファイル名のサフィックスにエンコードされる：-lh5- 等。 lhasa はすべての LZH メソッドを読める：

サフィックスアルゴリズム備考
-lh0-無圧縮 ヘッダにはメソッドが残る。
-lh1-LZ77 + 算術符号 (LZARI) オリジナルの LHarc メソッド。まれ。
-lh4-LZ77 + 静的 Huffman LZARI バリアント。
-lh5-LZHUF (LZ77 + 動的 Huffman) デフォルト。今見る .lzh の大半。
-lh6-静的 Huffman バリアント より高圧縮、ツールサポートは狭い。
-lh7-LZHUF + tree バリアント 実験的。
-lhx-LZH 互換マーカー 更新規約 + 分離された -lhd- ディレクトリヘッダ。
