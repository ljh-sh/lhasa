使い方

4 つのコマンド — その他は全部オプション。

使い方。4 つのコマンド — 一覧、展開、テスト、出力：

  lha l archive.lzh             # 一覧（ls）
  lha x archive.lzh             # 展開
  lha t archive.lzh             # テスト（CRC 検証）
  lha p archive.lzh file        # 標準出力へ表示（cat）

インストール：x eget ljh-sh/lhasa · 静的バイナリ、~200 KB、依存なし。
